# Protocols by Port

- **21**- FTP
- **22** - rsync (SSH)
- **23** - telnet
- **80** - http
- **443** - https
- **445** - SMP (For direct SMP communication)
- **873** - rsync (Daemon Mode) [Default]
- **3306** - MySQL

*** *You can also use [speedguide.net](http://speedguide.net) to discover ports of interest if no specific objective is present.*
